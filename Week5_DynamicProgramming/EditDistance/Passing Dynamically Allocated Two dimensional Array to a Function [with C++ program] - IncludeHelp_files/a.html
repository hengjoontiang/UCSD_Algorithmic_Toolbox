        Hi, Abhishek.

        Nice article.
        Short, to the point, and explains the ideas clearly.

        I'd like to share one more way of doing it, similar to the second method in your post.
        The sub-routine "show" could also be declared as follows:

        void show(int arr[], int x, int y)

        The rest of the sub-routine would stay the same.

        However, the call to the sub-routine would be slightly different:

        show(&A[0][0], n, m);

        I am sure there are other ways too, without going to the <vector> class (but using the <vector> class sure makes memory management a lot simpler!)
